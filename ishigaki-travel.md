# 石垣島旅行の知見

石垣島旅行へ行ったときの知見を書く。

# スケジュール

以下を考慮して6月末に旅行に行くことにした。

* 台風とかぶる確率が低い
* 梅雨明けで晴れる確率が高い
* ハイシーズン(7〜9月)を若干外すが、寒くない時期

# 事前にやっておくこと

* 航空券・ホテルの予約
  * ハイシーズンは人気のホテル・部屋は半年前から埋まりはじめるので、なるべく早めにおさえること！
* レンタカーの予約(プランに含まれていることも多い)
  * 石垣島ではタクシーも多く走っているが、レンタカーがあると行動の自由度が上がる。
* バスの予約
  * 和光市から空港、空港から和光市へのバスの予約をしておく。
  * バスは1ヶ月前の10:00から予約できる。
* レストランの予約
  * 人気店は予約必須
* アクティビティの予約
  * アクティビティは前日〜当日でも予約しやすいが、事前に予約しておくと安心。

# あると良い持ち物

* 日焼け止め、ラッシュガード、帽子など
  * 日傘は邪魔かと思って持っていかなかったが、あっても良かったかも。
* サンダルは足にしっかり固定できるタイプがオススメ。
  * ビーチは貝殻やサンゴが多く、素足だと痛い。しかし脱げやすいサンダルだとそのまま海に入れないので、脱げないタイプのサンダルがいい。
* 酔い止め
  * ツアーで船に乗ったりする場合はあるといい
* 水筒
  * フサキビーチリゾートではペットボトルのお水がない(ウォーターサーバー)ため、アクティビティ中に持ち運ぶ飲み物に困った。

# 石垣島の豆知識

* 羽田空港から石垣空港への直行便がある。直行便にするとやや値段が高くなるが、やはり楽なのでオススメ。
* 海開きは3月くらいにしている。
* 市街地に車を停めようとすると、駐車場探しに困ることがある。ホテルから市街地への移動ならタクシーを使ってもいいと思う。
* タクシーが多く走っているのでハイシーズンでも割とつかまる。
* ほとんどのお店で電子マネーが使えるため、現金はほぼ使わなかった。予備で5万円くらい持っておけば十分。
* フサキビーチリゾートでは各階に乾燥機があるので、濡れた服や水着はこれで乾かすと早い。ここで小銭がいる。
* 石垣島から竹富島までフェリーで15分で行ける。時間に余裕があれば竹富島観光も良さそう。

# 行ったところ・やったこと

* 川平湾、青の洞窟までシュノーケル、回復の滝 3大スポットツアー
  * 川平湾: 綺麗な景色を眺め、ビーチに降りて写真を撮って終わる。これくらいの短時間でちょうどいいかも
  * 青の洞窟: シュノーケルが楽しくて大満足！4歳の子供でもギリギリ楽しめた。青の洞窟は思ったより普通の洞窟。
  * 回復の滝: 森の中にある滝に入る。森の中は涼しく、お水に入ると寒い。短時間だけど楽しかった。
* 星空ツアー
  * 人気のないビーチに行って、満点の星空をバックに写真を撮るツアー
  * 天候に左右されやすいというのはあるが、綺麗な写真が撮れて思い出になった
* 底地ビーチ
  * それほどメジャーではないビーチにあえて行った。人も多すぎず、海も綺麗でよかった。
  * シャワーやトイレなどの設備は最低限といった感じだが不便はしなかった。
  * ビーチは木陰があって暑すぎず気持ちが良かった。
* 石垣鍾乳洞
  * それなりに大きい鍾乳洞で、中は涼しくて気持ちが良かった。あと意外と観光客が少なくて快適。
  * 時間に余裕があったら行ってもいいかも、というレベル。

# 旅行スケジュール(2泊3日)

2025/06/27(金) 〜 2025/06/29(日)

### 📅 1日目：到着から石垣の夜を満喫

✈️ 午前：石垣島へ出発、そして到着！

* **5:50** 和光市南口バス出発
* **7:00** 羽田空港第2ターミナル到着
* **8:30** ANA089便で羽田空港を出発
* **11:30** 石垣空港に到着
  * 石垣空港に到着したら、まずは空港内で昼食を済ませる。

🚗 午後：レンタカーでホテルに向かう

* **レンタカー貸出**：石垣空港からレンタカーのお店まではやや距離があるが、空港の入り口の前で送迎の車が待機しているのでそれに乗ればいい。

🏨 ホテルにチェックイン

宿泊先の「**フサキビーチリゾート ホテル＆ヴィラズ**」へ。
ホテルのプール、海があるので早速泳ぎに行く。

🍽️ 夜：レストランへ

* **18:00** お酒を飲みたいのでホテルからタクシーで移動し、「**鮨・島居酒屋じごろ**」でディナー
  * 夜でもタクシーはつかまりやすかった
* **20:45** 食事の後は、**夜空ツアー**に参加
  * 天候が良いときは満点の星空が見れて綺麗だった。ただ、帰りは23:00くらいになるので翌朝早いとちょっと疲れるかも。

## 🌴 2日目：マリンアクティビティとホテルステイを満喫

🌊 午前：アクティビティ

* **6:30** ホテルで朝食
  * 朝食は美味しかった。早い時間に行ったので大丈夫だったが、混雑すると待ちが発生するので注意。
* **8:00〜12:00** 川平湾、シュノーケリング、青の洞窟、などをまわるツアーに参加

🏊 午後：ホテルでリラックスタイム

* **13:00** ホテルのカフェで軽食をとり、休憩。
  * フサキ内には多くのレストランがあるが、昼食だと選択肢が1ヶ所しかなかった
* 午後は近くのビーチで貝殻拾いをして楽しんだ。

🍽️ 夜：レストランへ

* **17:30** ディナーは「**琉球新天地**」へ


## 🛫 3日目：最終日も石垣島を満喫

🌊 午前：地元のビーチへ

* **6:30** ホテルで朝食
* **ビーチへ行く**：観光客が多すぎない、少しマイナーなビーチを探して「底地ビーチ」へ
* **石垣島鍾乳洞へ**：時間が余ったので石垣島鍾乳洞へ

✈️ 午後：レンタカー返却、そして帰路へ

* **14:30** オリックスレンタカー新石垣空港ゆうな店でレンタカーを返却
* **15:35** 石垣島を15:35に出発
* **18:25** 羽田空港に到着
* **19:15** 羽田空港第2ターミナル7番のりば出発
  * バスの時間に間に合わず次の便(20:15)に変更した。飛行機がやや遅れたのと、空港到着から出るまでになんだかんだ時間かかった。
* **21:30** 和光市駅南口到着
