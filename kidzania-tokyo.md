# キッザニア東京 攻略

キッザニア東京は3〜15歳の子供が楽しめる職業体験型アトラクションです。
100種類もの職業、60種類ものパビリオンがあり、子供だけでなく見ている大人も大満足できる施設なのです。

しかしそんなキッザニアは事前調査なしで行くと「誰もやりたがらない不人気な職業しか体験できなかった」「やることがなくなりベンチに座ってボーッとする時間(無職)がうまれた」など、情弱が食い物にされる、まさに社会の縮図とも言うべき恐ろしい場所なのです。

ここではそんなキッザニアを100%楽しむためにやるべきことをまとめます。

## 入場したらこのサイクルをひたすら回す

キッザニアで重要な「予約」と「受付」の概念を理解してスケジュールを組むことが最重要となる。基本は以下のサイクルを回すイメージで良い。

1. まずお仕事を予約する
    * 「絶対にやりたいお仕事」で「人気で直近の時間が空いていない」ものを優先して予約する。
    * 1つ予約すると次の予約が取れなくなるため注意。
2. 予約時間までの間にできるお仕事に受付する
    * 次に優先度の高いお仕事で、できればすぐ(5〜15分以内)にお仕事開始できるものを受付する
    * 1つ受付するとそのお仕事が完了するまでは他のお仕事ができなくなるので注意！
    * ここで例えば「1時間先のお仕事を予約」してしまうと、1時間暇になってしまう。
    * また、お仕事によっては時間が長いものがあるので、予約しているお仕事の時間に被ってしまわないかを確認すること。
3. 受付したお仕事開始！親は全力で子の写真を撮る
    * お仕事が終わりそうになったら、他のお仕事の空き状況をチェックして次の予約・受付するお仕事の目星をつけておく。
    * 予約したお仕事の開始までまだ時間がある場合は、間にやれる他のお仕事をする。
4. 予約したお仕事開始！
5. 予約枠が空く。1 に戻る。

# 知っておきたいキッザニアの基礎知識

* キッゾ
    * キッザニア内で使えるお金。
    * 入場するときに50キッゾもらえる。
    * お仕事をするとキッゾがもらえるが、お客さんとして参加する場合は逆にキッゾを払わなくてはならない。でもキッゾが不足することは基本ないので気にせずやりたいお仕事をやればいい。
    * 銀行口座を作るとキッゾを入れておくことができる。またお金をおろすだけならATMからもできる。
    * 三越でキッゾを使ってお買い物ができるが、キッザニアに1,2回行ったくらいではたいしたもの買えない。経験上楽しいかも。
* 写真・動画
    * 警察官、消防士、パティシエ、パイロット、キャビンアテンダントなどは体験中に写真を撮ってくれて、あとで購入することができる。
    * CMタレントは撮影したCMを動画データとしてあとでダウンロードできる。
    * 写真を購入するとデジタルデータでももらえる。また、アクリルスタンドやキーチェーンなどに加工してもらうこともできる。
* 服装・髪型
    * 館内は暖かく、頻繁に着替えるので、薄着のシャツ、ズボンでいるといい。
    * 髪型は可愛くしていっていいと思うが、帽子やヘルメットを被ったりするのであまり凝ったことはできないかも。
* キッザニア内のWi-Fiがある。当日受付するとパスワードを書いた紙がゲットできる。
    * ただ、開始直後はアクセスが殺到するためWi-Fiが使い物にならないことがある。臨機応変にモバイルデータ通信に切り替える。
* 一緒に回りたい子がいるなら事前に何をやるか打ち合わせしておくといい。あと一緒に回る子は2,3人が限界だと思う。「残り1枠」みたいなのも多いので、人数は少ないほうが入りやすい。
